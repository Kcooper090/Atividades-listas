O que é um algoritmo?

Um manual limitado de ações que devem ser executadas para uma finalidade exigida.

Entrada, processamento e saída

Entrada: Um meio para solicitar alguma ação do programa. Exemplos: mouse, teclado, qualquer dispoistivo que faça contato com o processado.


Processamento: Meio que irá executar o pedido da entrada. Exemplo: Placa mãe, memória e principalmente processador


Saída: Parte que irá apresentar o resultado da execusão feita pelo processador e requerida pela entrada. Exemplo: Monitor, caixa de som, tela, impressora.

Estrutura sequencial na programação

é o formato mais básico de programação, onde as instruções são executadas em ordem linear, de cima para baixo, uma após a outra

Variável. Exemplo

é um espaço reservado na memória do computador, com um nome simbólico, usado para armazenar dados que podem ser modificados durante a execução de um programa


Inteiro (int): Armazena números inteiros. Ex: Idade, número de casa, quantidade de objetos


Ponto Flutuante (float/double): Armazena números decimais. Ex: Altura, peso, salário, temperatura, quilômetragem, coordenadas, probabilidade, velocidade, notas


Caracter ou cadeia - Texto. Ex: nome, endereço, lista


Bolean - Valor verdadeiro ou falso.

3 tipos de dados usados na programação e exemplos

informações estruturadas (números, texto, booleanos) que computadores processam, armazenam e manipulam para realizar tarefas

Inteiros (int): Números inteiros, positivos ou negativos, sem decimais (ex: 10, -5). Ponto Flutuante (float/double): Números reais com casas decimais (ex: 10.5, 3.14). Caracteres (char): Um único símbolo, letra ou número (ex: 'A', '2', '#').

O que são operadores? exemplo

Operadores são símbolos que realizam operações em dados (operandos). Os principais tipos são: aritméticos ex: -, +. e os relacionais == .

Operadores relacionais

Operadores relacionais são símbolos utilizados na programação para comparar dois valores ou expressões, retornando um resultado booleano: Verdadeiro (true ou 1) ou Falso (false ou 0).

Valor booleano

é um tipo de dado lógico na computação que representa apenas dois estados possíveis: verdadeiro (true/1) ou falso (false/0)

Expressão lógica

Uma expressão lógica (ou booleana) é uma proposição matemática que utiliza operadores lógicos (AND, OR, NOT) para avaliar condições e retornar um valor exclusivo: Verdadeiro (1/True) ou Falso (0/False).

Fluxograma

é a representação gráfica de um processo, utilizando símbolos padronizados (como retângulos, losangos e setas) para mapear etapas, decisões e o fluxo de trabalho de forma clara

pseudocódigo

forma genérica e simplificada de escrever algoritmos utilizando linguagem natural (como português) estruturada, sem seguir regras rígidas de sintaxe de linguagens de programação

Diferença entre algaritmo e programa

Um algoritmo é uma sequência lógica de passos (receita) para resolver um problema, independente de linguagem. Um programa é a implementação prática desse algoritmo em uma linguagem de programação específica (como Python ou C++) que um computador consegue executar. Algoritmos são conceituais; programas são concretos.

Erro lógico

ocorre quando um programa de computador é executado sem travar, mas produz resultados incorretos ou indesejados

Por que lógica de programação é importante?

é fundamental para estruturar o pensamento de forma sequencial e sistemática, permitindo a criação de algoritmos eficientes para resolver problemas complexos

Por que o algotritmo precisa ser escrito em ordem?

porque ele é definido como uma sequência finita de passos lógicos e bem definidos para realizar uma tarefa ou resolver um problema

SE não tem água OU não tem café:
ENTÃO Comprar ingredientes.
RETORNAR ao passo 2.
SENÃO (Tem tudo):
Aquecer a água: Colocar água para esquentar até ~94ºC.
Preparar o filtro: Colocar o filtro de papel/pano no suporte.
Pré-aquecer (Opcional): Jogar água quente no filtro para escaldar e retirar gosto de papel.
Adicionar café: Colocar o pó de café no filtro (proporção sugerida: 1g para cada 15ml-18ml de água).
Extração (Infusão): Despejar a água quente lentamente sobre o pó em movimentos circulares.
Filtração: Aguardar a água passar pelo café.
Verificar saída: O café está na xícara?
