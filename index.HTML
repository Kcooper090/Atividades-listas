Questionário 2 - Versionamento

O que é o Git?

Um sistema de versões compartilhado

Para que serve o GitHub?

Para o compartilhamento de programas ou códigos para versionamento

Para que serve o Git clone?

Fazer uma cópia para o repositório local do Git

Para que serve o git add?

Adiciona novos arquivos ao inventário

Para que serve o comando Git commit?

Um save point que registra alterações específicas no código

Para que serve o comando Git Push?

envia os commits realizados no repositório local para um repositório remoto (como GitHub, GitLab ou Bitbucket), atualizando o histórico compartilhado
